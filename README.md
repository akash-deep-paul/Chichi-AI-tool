# Chichi-AI-tool

Chichi is a Python-based voice-controlled desktop assistant that listens for spoken commands and performs tasks like opening YouTube videos, searching Google, opening Wikipedia pages, telling time/date, launching applications, and performing basic calculations.

The project was built using free Python libraries without paid APIs and works directly from the terminal.

---

## Features

- Voice command recognition
- Custom trigger-word detection ("Chichi")
- YouTube search and autoplay
- Google search support
- Wikipedia page opening
- Time and date responses
- Basic calculator
- Open desktop applications
- Close browser windows
- Text-to-speech voice responses
- Multiple trigger-word variations support

---

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- requests
- sounddevice
- numpy
- webbrowser
- Regular Expressions (re)
